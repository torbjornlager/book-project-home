A calculator

The calculator statechart models a simple four-function calculator as a hierarchy of states that track whether a first or second operand is being entered, whether it begins with a sign or a decimal point, and so on. Events are individual key presses: digits, operators, equals, decimal point, and clear. The machine sends a display/1 output message whenever the display value changes, so you can see the result in the terminal.

Spawn the machine. The Examples menu will be loaded with a sequence that computes 3 + 4 = 7: 

?- statechart_spawn(Pid, [
       load_uri('/examples/statecharts/calculator.xml'),
       trace(true)
   ]).

Send key events from the Examples menu one at a time. A display/1 message appears in the terminal after each one that changes the display:

$Pid ! digit(3) enter the digit 3
$Pid ! oper(plus) press +
$Pid ! digit(4) enter the digit 4
$Pid ! equals evaluate — display shows 7
$Pid ! c clear everything and reset
Other operators. oper(minus), oper(star), and oper(div) send −, ×, and ÷. point enters a decimal point. ce clears the current entry without resetting the whole expression.
